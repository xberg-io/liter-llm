# liter-llm - Dart Bindings

Universal LLM API client with Rust-powered polyglot bindings.

## Installation

Add to `pubspec.yaml`:

```yaml
dependencies:
  liter_llm:
    git: <repo-url>
```

## Quick Start

```dart
// Phase 2: Dart bindings via flutter_rust_bridge. Skeleton only.
```
