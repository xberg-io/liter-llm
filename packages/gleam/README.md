# liter_llm

Universal LLM API client with Rust-powered polyglot bindings.

## Building

Install Gleam (see [gleam.run](https://gleam.run)):

```sh
gleam build
gleam test
```

## Usage

Add to your `gleam.toml`:

```toml
[dependencies]
liter_llm = "{github = \"example/liter_llm\"}"
```
