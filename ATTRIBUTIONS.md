The provider configuration table in `schemas/providers.json` was derived from
LiteLLM's provider endpoint support data, model registry, and OpenAI-compatible
provider configurations. This data was used as a one-time bootstrap to seed
our provider registry with base URLs, authentication patterns, endpoint support,
and model prefix mappings for 163 LLM providers.
